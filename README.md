# Tic Tac Toe


By [Leandra Tejedor](https://github.com/ltejedor).


## Description
**Tic-Tac-Toe**

Play against your friends, and see who's the best at putting three symbols in a row.
Computer Player in the process of getting summoned.


To play, run

```console
cd tic-tac-toe
ruby tic_tac_toe.rb
```
